로그인 세션스토리지에 저장

진료과/의사 소개 :조회
실시간 예약 기능 구현

예약 : 먼저 진료예약 페이지로 이동 -> 진료과와 진료일을 선택 후 오른쪽에 데이터를 랜더링함 -> 만약, 로그인을 안했을 경우? 로그인페이지로 이동 -> 로그인 완료 후 데이터를 입력하고 예약하기 버튼을 클릭 ->

관리자로그인 후 관리자 페이지로 이동 -> 

-1 관리자 : 의사별 담당환자 리스트
-2 총 게시글 수 조회하는 쿼리
-3 의사별 담당환자 차트
-2 예약취소 : SCH_STATUS='N'로 변경 (상태변경)
*1 환자가 보는 나의 예약페이지
1. 의사 진료과 불러오기
2. 예약유무 확인 (진료과와 날짜를 선택했을때)
3. 예약 insert
4. 로그인안했으면 로그인페이지로 이동
